Worked Examples

This page walks through four real-world protection coordination scenarios from start to finish. Each example sets up a transformer, configures relays, and examines the resulting trip times and coordination margins. Use them as templates for your own studies or to build familiarity with the simulator.

Example 1 -- Basic 33/11 kV Substation

A straightforward two-relay coordination study on a medium-voltage distribution transformer. The goal is to grade the primary and secondary overcurrent relays so that the secondary relay trips first for downstream faults, with an adequate coordination time interval (CTI) to the primary.

Scenario

Parameter Value
Transformer rating10 MVA
Primary voltage33 kV
Secondary voltage11 kV
Impedance (Z%)8%

Relay Configuration

Relay Side Curve CT Primary Pickup Mul. TMS
R1 -- Primary OC Primary IEC Standard Inverse 200 A 0.50 0.40
R2 -- Secondary OC Secondary IEC Standard Inverse 600 A 0.50 0.15

Fault Current Derivation

Using the formula I_fault = (MVA x 1000) / (kV x sqrt(3) x Z%/100):

  • Secondary fault current (100%): (10 x 1000) / (11 x 1.732 x 0.08) = 6561 A
  • Primary fault current (100%): (10 x 1000) / (33 x 1.732 x 0.08) = 2187 A

Pickup Settings (Iset)

  • R1: Iset = 200 x 0.50 = 100 A
  • R2: Iset = 600 x 0.50 = 300 A

Trip Times at 100% Fault Level

Using the IEC Standard Inverse formula t = (0.14 / ((I/I_set)^0.02 - 1)) x TMS:

  • R2 (secondary): I/Iset = 6561/300 = 21.87. Trip time = (0.14 / (21.87^0.02 - 1)) x 0.15 = approximately 0.34 s
  • R1 (primary): I/Iset = 2187/100 = 21.87. Trip time = (0.14 / (21.87^0.02 - 1)) x 0.40 = approximately 0.91 s

CTI Check

The coordination time interval between R1 and R2 is 0.91 - 0.34 = 0.57 seconds. This exceeds the typical minimum CTI of 0.3 to 0.4 seconds, confirming that R2 will always clear downstream faults before R1 operates. The grading is satisfactory.

Key Observations

  • Both relays use the same curve type (IEC SI), so the grading is achieved purely through TMS differentiation.
  • The CT ratios are chosen to match the full-load current on each side, with a pickup multiplier of 0.5 providing sensitivity to faults at roughly half the CT primary rating.
  • The 0.57 s CTI margin gives comfortable room for breaker operating time and relay timing tolerances.

Example 2 -- Full Coordination with Cable Protection

This example extends Example 1 by adding two additional relays and enabling the cable damage overlay. The goal is to verify that all four relays coordinate with each other and that the secondary cable is protected within its thermal damage limit.

Scenario

Same transformer as Example 1: 10 MVA, 33/11 kV, Z = 8%. The secondary side uses Cu/XLPE cable with a cross-section of 95 mm2.

Relay Configuration

Relay Side Curve CT Primary Pickup Mul. TMS
R1 -- Primary backup Primary IEC Standard Inverse 200 A 0.50 0.40
R2 -- Primary main Primary IEC Standard Inverse 200 A 0.50 0.30
R3 -- Secondary backup Secondary IEC Standard Inverse 600 A 0.50 0.20
R4 -- Secondary main Secondary IEC Standard Inverse 600 A 0.50 0.10

TMS Grading

The four relays are graded with descending TMS values from the slowest (R1, furthest upstream) to the fastest (R4, closest to the fault). At 100% fault level, the approximate trip times are:

  • R4: TMS 0.10 -- approximately 0.23 s
  • R3: TMS 0.20 -- approximately 0.45 s (CTI to R4: ~0.22 s)
  • R2: TMS 0.30 -- approximately 0.68 s (CTI to R3: ~0.23 s)
  • R1: TMS 0.40 -- approximately 0.91 s (CTI to R2: ~0.23 s)

Cable Damage Verification

Enable the cable damage overlay with these settings:

  • Conductor: Copper
  • Insulation: XLPE (k = 143)
  • Cross-section: 95 mm2

At the maximum secondary fault current of 6561 A, the cable damage time is:

t = (143 x 95 / 6561)^2 = (13585 / 6561)^2 = 2.07^2 = 4.29 s

All four relays trip well under 4.29 seconds at this current, so the cable is protected. Check the TCC chart to confirm that the relay curves remain below the cable damage line across the entire fault current range, not just at the maximum value.

Key Observations

  • With four relays, the CTI between adjacent relays is tighter (~0.22 s). In practice you may want to increase the spacing by widening the TMS steps.
  • The cable damage curve at 95 mm2 Cu/XLPE sits comfortably above all relay curves, confirming adequate cable protection.
  • If the cable were smaller (e.g., 25 mm2), the damage curve would drop significantly, and you would need to verify that R4 still trips below it.

Example 3 -- MCB Coordination

In small distribution transformers, the secondary side often feeds circuits protected by miniature circuit breakers. This example demonstrates how an upstream IDMT relay on the primary side coordinates with a downstream MCB on the secondary.

Scenario

Parameter Value
Transformer rating0.5 MVA
Primary voltage11 kV
Secondary voltage0.415 kV
Impedance (Z%)5%

Device Configuration

Device Type Details
MCB Type C, 32 A Secondary side -- magnetic trip at 5-10x (160-320 A)
Relay R1 IEC Standard Inverse Primary side -- CT 50 A, pickup 0.80, TMS 0.30

Fault Current Derivation

  • Secondary fault current (100%): (0.5 x 1000) / (0.415 x 1.732 x 0.05) = 13,908 A
  • Primary fault current (100%): (0.5 x 1000) / (11 x 1.732 x 0.05) = 525 A

Relay Pickup

R1: Iset = 50 x 0.80 = 40 A. At 100% fault level on the primary, I/Iset = 525/40 = 13.13.

Coordination Check

Enable the MCB overlay with Type C, 32 A. On the TCC chart, the MCB curve appears in the low-current region (up to a few hundred amps on the secondary axis). The relay curve, referred to the primary side, should sit above the MCB band at all overlapping current levels.

The Type C 32 A MCB has:

  • Thermal region: Inverse-time trip from approximately 1.13x to 5x rated current (36 A to 160 A)
  • Magnetic region: Instantaneous trip between 5x and 10x rated current (160 A to 320 A)

At the MCB's magnetic trip threshold (320 A on the secondary), the relay on the primary sees a proportionally referred current. The relay trip time at this current level should exceed the MCB's maximum clearing time plus the CTI margin.

Key Observations

  • The MCB clears low-level faults near the secondary distribution board without involving the upstream relay.
  • For high-level faults close to the transformer, both the MCB (instantaneously) and the relay respond, but the MCB acts faster due to its magnetic trip.
  • The relay serves as backup protection -- it operates only if the MCB fails to clear the fault.
  • The relatively low TMS of 0.30 keeps the relay fast enough to protect the transformer, while still sitting above the MCB band.

Example 4 -- IEEE Curve Comparison

This example places three relays on the same transformer secondary, each using a different IEEE curve type with identical pickup and TMS settings. The purpose is to illustrate how the choice of curve shape affects trip times at different fault levels.

Scenario

Parameter Value
Transformer rating10 MVA
Primary voltage33 kV
Secondary voltage11 kV
Impedance (Z%)8%

Relay Configuration

Relay Side Curve CT Primary Pickup Mul. TMS
R1 Secondary IEEE Moderately Inverse 600 A 0.50 0.20
R2 Secondary IEEE Very Inverse 600 A 0.50 0.20
R3 Secondary IEEE Extremely Inverse 600 A 0.50 0.20

All three relays share the same CT primary (600 A), pickup multiplier (0.50), and TMS (0.20), so Iset = 300 A for each. The only difference is the curve type.

Trip Time Comparison

The following table shows approximate trip times at selected fault levels. The secondary fault current at 100% is 6561 A.

Fault Level Ifault (A) I/Iset R1 (IEEE MI) R2 (IEEE VI) R3 (IEEE EI)
20% 1312 4.37 1.07 s 0.77 s 0.59 s
50% 3281 10.94 0.57 s 0.20 s 0.06 s
80% 5249 17.50 0.47 s 0.12 s 0.02 s
100% 6561 21.87 0.43 s 0.10 s 0.01 s

Key Observations

  • At low fault multiples (I/Iset near 4x): All three curves produce broadly similar trip times, with the EI curve only slightly faster than MI.
  • At high fault multiples (I/Iset above 10x): The curves diverge dramatically. IEEE EI trips in tens of milliseconds, while IEEE MI remains above 0.4 seconds. This is the defining characteristic -- "extremely inverse" curves are much steeper.
  • IEEE MI is suitable when you need a relatively flat characteristic that does not change trip time drastically with current magnitude.
  • IEEE VI provides a middle ground with moderate steepness, useful for grading with fuses or other inverse-time devices.
  • IEEE EI is ideal for protecting equipment with I^2 x t damage characteristics (such as cables and transformers), because the relay curve shape closely matches the damage curve slope.

Open this study in the simulator to see all three curves on the same TCC chart. Adjust the fault level slider and observe how the trip times change at different points along each curve.
