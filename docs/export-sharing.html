Export & Sharing

The simulator provides several ways to save, export, and share your relay coordination studies. Everything runs locally in your browser -- no server uploads, no accounts required.

PDF Export

Click the PDF button to generate a print-friendly view of your current study. The PDF output includes:

  • A high-resolution black-and-white TCC (time-current characteristic) chart, optimized for print
  • Transformer parameters (MVA, primary/secondary kV, impedance)
  • Relay settings table with all columns: side, CT primary, pickup multiplier, TMS, definite-time pickup, DT delay, curve type, and label
  • Trip time table showing calculated trip times at every fault level from 10% to 100%
  • CTI coordination margins table between adjacent relay pairs
  • Overlay curves table (if any overlay curves are enabled)
  • Remarks field content

Use your browser's Print or Save as PDF dialog to save the output. If you have filled in the Report Settings fields (company name, project reference, etc.), these appear in the report header for a professional-quality document.

CSV Export

Click the CSV button to download a comma-separated values file containing the fault level data from your study. The CSV columns are:

  • Fault % -- the fault level as a percentage of full-load current
  • Primary Fault (A) -- the calculated primary-side fault current in amperes
  • Secondary Fault (A) -- the calculated secondary-side fault current in amperes
  • Per-relay columns for Trip Time (s) and Fault Current (A)

Import the CSV into Excel, Google Sheets, or MATLAB for further analysis, custom charting, or integration with other protection coordination tools.

URL Sharing

Click the Share button to copy a link containing your full simulator configuration encoded directly in the URL. Send this link to colleagues via email, chat, or any other channel.

When the recipient opens the link, the simulator loads with your exact settings: all relay parameters, transformer data, overlay selections, fault level position, and remarks. The link is entirely self-contained -- no account or login is needed to view it.

After loading a shared URL, the configuration parameter is automatically cleaned from the browser address bar to keep the URL tidy.

Named Studies

Save your relay configurations as named studies for quick retrieval later:

  • Save: Click "Save Current" in the Studies panel and enter a descriptive name for the study
  • Load: Click a study name in the list to restore that full configuration
  • Delete: Click the X button next to a study to remove it
  • Export: Click "Export All" to download every saved study as a single JSON file
  • Import: Click "Import JSON" to load studies from a previously exported file -- useful for backups or sharing multiple configurations between machines

Studies are stored in your browser's local storage. They persist across sessions but are specific to the browser and device you are using. Use Export/Import to transfer studies between devices.

Report Settings

The Report Settings section provides optional fields that appear in the header of PDF exports:

  • Company Name
  • Project Reference
  • Document Reference
  • Revision

Fill these fields in before exporting to generate professional reports with proper document identification. These values are also saved as part of your study configuration when using Named Studies or URL Sharing.

Offline Use (PWA)

The simulator works offline after your first visit. A service worker caches all required assets so that the full application is available without an internet connection.

You can install the simulator as a standalone app on your device using your browser's "Install" or "Add to Home Screen" option. Once installed, it launches like a native application with its own window and icon.

All calculations run entirely in your browser -- no data is ever sent to any server. Your relay settings, studies, and report preferences remain private on your device.
